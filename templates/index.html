Bem-vindo ao PyStorage!

Este sistema foi desenvolvido visando fixar os conteúdos sobre Python e Banco de Dados da Residência em TIC 55, utilizando o framework Flask como servidor web com integração ao banco de dados PostgreSQL.

Os conceitos de API, seus endpoints e métodos HTTP foram grandemente abordados nesse projeto.

Todas as chamadas ao banco de dados são feitas através de queries puras, sem o uso do SQLAlchemy.

Na raiz do projeto existe o arquivo db_script.sql para criar o banco de dados e suas tabelas correspondentes.

Nesse sistema básico você é capaz de:

  • Cadastrar e organizar categorias.
  • Cadastrar e organizar produtos.

Este é um projeto de estudos somente :) ...
